# Badge Generator

Generate a customizable GenAI-Ethics badge with your studio logo and policy summary.

## Usage

1. Install the required dependencies: `poetry install`
2. Run the badge generator: `python src/badge_generator.py --studio-logo <studio_logo> --policy-summary <policy_summary>`
3. Verify the badge: `python src/badge_generator.py --studio-logo <studio_logo> --policy-summary <policy_summary> --verify`

## Tests

Run the tests: `pytest tests/test_badge_generator.py`
